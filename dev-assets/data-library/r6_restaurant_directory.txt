============================

- DETERMINISTIC: Data verified within last 4 hours, maximum reliability. Agent can answer with full confidence.
- PROBABILISTIC: Data verified within 24 hours, good reliability. Agent should mention verification time.
- DEGRADED: High reliability score but data older than 24 hours. Agent should note data staleness.
- SPECULATIVE: Low reliability score. Agent should suggest confirming with restaurant.
- STALE: Data older than 7 days. Agent must NOT answer. Offer human handoff immediately.
